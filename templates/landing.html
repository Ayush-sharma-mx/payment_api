🔌 Interactive Endpoints & cURL Reference

POST /api/payments/process-payment/
Core Engine

Processes a transaction atomically using SHA-256 fingerprinting and SELECT ... FOR UPDATE locks. Replays identical keys instantly without re-processing.

curl -X POST "{{ host_url }}/api/payments/process-payment/" -H "X-API-KEY: {{ demo_api_key }}" -H "Idempotency-Key: test-uuid-9876" -H "Content-Type: application/json" -d '{"amount": "49.99", "currency": "USD", "payer_email": "reviewer@example.com", "description": "Reviewer Demo Charge"}'
GET /api/payments/

Lists historical payment records, their timestamps, amounts, and completion statuses.

curl -X GET "{{ host_url }}/api/payments/" -H "X-API-KEY: {{ demo_api_key }}"
GET /api/ai/events/
✨ AI Intelligence

Read-only historical ledger capturing every transaction attempt, PII-scrubbed request fingerprints, and AI explanations distinguishing benign retry storms from polling loops.

curl -X GET "{{ host_url }}/api/ai/events/" -H "X-API-KEY: {{ demo_api_key }}"
GET /api/ai/risk-scores/
✨ AI Intelligence

Hybrid risk scoring engine combining deterministic weighted rules (amount, retry velocity) with LLM pattern adjustments clamped strictly within ±0.1.

curl -X GET "{{ host_url }}/api/ai/risk-scores/" -H "X-API-KEY: {{ demo_api_key }}"
POST /api/ai/query/
✨ AI Intelligence

Sandboxed natural language to SQL query engine with static safety validation (enforces SELECT-only, whitelisted tables, and LIMIT <= 500).

curl -X POST "{{ host_url }}/api/ai/query/" -H "X-API-KEY: {{ demo_api_key }}" -H "Content-Type: application/json" -d '{"question": "How many duplicate payment attempts were detected today?", "auto_execute": true}'
